Introduction to Sensitivity Analysis#

This site is still under construction

If you’re wondering what the deal with Sensitivity Analysis, is, please follow tutorial in the Jupyter notebook here. After completing this tutorial, you will be able to:

  1. Install and use basic Python sensitivity analysis packages

  2. Understand the uses and limitations of different sensitivity analysis methods

  3. Implement local and global sensitivity analysis methods

  4. Understand the differences in the mathematical formulations of these methods

  5. Apply sensitivity analyis to a small dataset

In this tutorial, the dataset you will be using consists of two files :

  1. Objectives_f125.csv

  2. financial_model_DVs.csv

Please ensure that both these files are in the same folder as your Jupyter notebook before beginning. Once you’ve completed this tutorial, here are some other resources to help you advance your sensitivity analysis skills:

  1. The MSD UC Ebook

  2. Sensitivity analysis tools in different computing languages

  3. Sensitivity analysis within the context of MORDM

  4. Milton Freedman’s thermostat

  5. Information theory and moment-independent sensitivity indices

  6. Sampling for sensitivity analysis

  7. Visualizing Sobol sensitvity indices using radial plots

  8. Magnitude-varying sensitivity analysis Part 1 and Part 2
